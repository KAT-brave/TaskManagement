# 機能要件

## 1. 認証機能

- ユーザー登録（名前・メールアドレス・パスワード）
- ログイン / ログアウト
- 自分のデータのみ表示・操作できる

## 2. ボード機能

- ボードの作成・編集・削除
- ボード一覧の表示

## 3. リスト機能

- リストの作成・編集・削除
- ボード内に複数のリストを表示

## 4. カード機能

- カードの作成・編集・削除
- タイトルと説明文・期限日を持つ

## 5. カード移動機能

- ドラッグ&ドロップでリスト間を移動できる

## 7. ユースケース / 操作フロー

### タスクを作成して完了にするまでの流れ

1. ログイン画面からログインする
2. ボード一覧画面で対象のボードをクリックする
3. 「Todo」リストの「+ カード」をクリックする
4. タイトルと説明を入力して保存する
5. 作成されたカードをドラッグして「進行中」リストに移動する
6. タスクが完了したらカードを「完了」リストにドラッグして移動する

### ボードを新しく作る流れ

1. ボード一覧画面で「+ 新規作成」をクリックする
2. ボード名を入力して保存する
3. 作成されたボードの詳細画面に遷移する
4. 「+ リスト追加」でリストを作成する

### ゲストとしてログインする流れ

1. ログイン画面の「ゲストログイン」ボタンをクリックする
2. デモアカウントで自動ログインされる
3. ボード一覧画面に遷移する
