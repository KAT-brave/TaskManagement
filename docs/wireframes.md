# タスク管理アプリ ワイヤーフレーム

## 1. トップ画面（/）

```
+------------------------------------------+
|         TaskManagement                   |
|                                          |
|      タスクをシンプルに管理しよう        |
|                                          |
|        [新規登録]  [ログイン]            |
|                                          |
+------------------------------------------+
```
